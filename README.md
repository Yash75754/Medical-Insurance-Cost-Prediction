🌟 Project Overview

Welcome to the Medical Insurance Cost Prediction System 🚀

This project is a smart Machine Learning web application that predicts a person's medical insurance cost based on several health and lifestyle factors such as:

✅ Age
✅ BMI
✅ Smoking Habit
✅ Gender
✅ Region
✅ Number of Children

The system uses a trained Random Forest Regression Model to analyze the data and estimate insurance expenses accurately.

The project is built with:

🐍 Python
🎨 Streamlit
🤖 Machine Learning
📊 Data Visualization
✨ Why This Project is Special?

This project is not just a simple prediction model.

It combines:

🔥 Machine Learning
🔥 Interactive UI
🔥 Beautiful Data Visualization
🔥 Real-world Healthcare Use Case
🔥 User-Friendly Dashboard

The goal is to make insurance prediction simple, fast, and visually understandable.
